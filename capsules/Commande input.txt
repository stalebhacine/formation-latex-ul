Un document LaTeX comporte parfois des tableaux ou des figures
complexes dont le code source est très long. Deux documents peuvent
aussi partager des parts significatives de code source.

Dans un cas comme dans l'autre, on peut simplifier son travail en
insérant dand un document le contenu d'un autre fichier avec la
commande 'input'.

* * *

La commande 'input' prend en argument le nom d'un fichier sans
l'extension '.tex'. LaTeX va insérer le contenu de ce fichier à
l'endroit où la commande apparait. Pour LaTeX, c'est exactement comme
si le contenu du fichier tierce se trouvait dans le fichier de départ.

Nous allons illustrer l'utilisation de la commande 'input' par deux
exemples.

* * *

[Intertitre: Exemple 1 - Insertion d'un long tableau]

La mise en forme d'un long tableau peut nécessiter beaucoup de code
source, rendant du coup l'édition d'un document un peu moins
conviviale.

Et si l'on pouvait «cacher» le bloc de code du tableau?

Certains éditeur de texte permettent de faire cela, mais LaTeX fournit
une solution par le biais de la commande 'input'.

On sauvegarde le code du tableau dans un fichier externe au fichier
.tex principal, puis l'on remplace le code du tableau par la commande
pour insérer en lieu et place le fichier externe.

À la compilation, rien n'y parait!

* * *

[Intertitre: Exemple 2 - Partage de contenu entre plusieurs fichiers]

Il est permis d'utiliser la commande 'input' dans le préambule d'un
document. La commande peut donc s'avérer très pratique pour utiliser
un même préambule dans plusieurs documents.

Dans le cadre d'un cours à l'École d'actuariat de l'Université Laval,
j'ai produit quatre documents qui partagent le même style d'ensemble:
mise en page, polices de caractères, commandes et environnements
spéciaux, etc.

Pour éviter de devoir apporter les éventuelles modifications aux
préambules de chacun des documents, j'ai placé la partie commune des
préambule dans un fichier séparé. Ensuite, le fichier principal de
chaque document débute par le chargement du préambule commun.

Non seulement cela permet-il d'assurer l'uniformité entre les
documents, mais les modifications sont très rapides à faire.

* * *

La commande 'input' est très simple à utiliser. Vous découvrirez
sûrement d'autres usages selon vos besoins!
